EpisoPass - 関連システム
パスワードに関する問題を解決しようとする関連システムについて説明します。
ここで紹介しているあらゆるパスワード管理システムは管理用パスワードを必要としており、それを覚えておく必要があります。 EpisoPassはパスワードを全く覚える必要がありません。
  • 1Password
    • 1PasswordはAgileBits社が販売しているパスワード管理システムです。 iPhoneアプリが1600円と高価ですが広く利用されています。 複数のパスワードを管理するために「マスターパスワード」を利用します。

  • LastPass
    • LastPassはLastPass社が販売しているパスワード管理ソフトです。 LastPassもマスターパスワードを利用して複数のパスワードを管理します。

  • Password Safe
    • Password Safeはオープンソースのパスワード管理データベースです。 マスターパスワードを利用します。

  • KeePass
    • KeePassはPasswordSafeをもとにしたフリーのパスワード管理ソフトです。

  • ミルパス
    • ミルパスはキングジムが販売しているパスワード記憶文房具です。

  • ノートンIDセーフ
    • ノートンIDセーフもパスワード管理ソフトです。

  • Dashline
    • Dashlineもパスワード管理ソフトです。

  • ID Manager
    • ID ManagerはWoodenSoldier氏の開発した フリーのパスワード管理ソフトです。 マスターパスワードを利用します。
