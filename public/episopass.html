概要

詳細はEpisoPass情報ページに記述してあります。
  • EpisoPass™ は、個人的な記憶からパスワードを生成するシステムです。
  • たとえばこのページの問題に正しく答えるとAmazonのパスワードが生成されます。
  • 上の「問題編集」で問題を作成し、「HTML作成」でパタンを指定してから「HTMLダウンロード」をクリックすると、 パスワードを生成するための上記のようなHTML(生成HTML)をダウンロードできます。
  • EpisoPassはepisopass.htmlという単一HTMLファイルで動作するので、 ダウンロードしてローカルマシンのブラウザで動かすことができます。 ネットワークやサーバは必要ありません。
  • 編集時のデータはどこにも保存されませんしどこにも送られません。
  • 生成HTMLもどこでも動かすことができます。安全な場所に保存しておくと良いでしょう。
  • 生成HTMLを問題編集画面にDrag&Dropすると問題データを再編集できます。

EpisoQ

  • episoqを利用すると問題を簡単に作成できます。
  • episoqは npm install -g episoq でインストールできます。

ブラウザ拡張機能

  • ブラウザ拡張機能を利用するとEpisoPassをブラウザのログイン画面で使えます。
  • サービス名_アカウント名.html という名前の生成HTMLにアクセスすると問題データがブラウザに保存され、 そのデータが拡張機能から参照可能になります。

問題作成支援

人名や地名のリストから簡単にEpisoPass問題を作ることができます。 名前と質問のリストを入力して ボタンを押して下さい。
  • 名前リスト
    • 他人が知らない思い出がある人名や地名を30行ぐらい並べて下さい。
    • 適当な名前を混ぜてもかまいません。
  • 質問リスト
    • 上のリストのうち10個程度に対し、それを特定できる質問を並べて下さい。
    • 他人が知らない秘密の事実で、将来忘れないようなエピソードを書いて下さい。
