EpisoPass - 記憶からパスワードを生成
EpisoPassは、パスワードを考えたり覚えたりする苦労を解消するシステムです
New! なぞなぞを簡単に作成できるシステムを作りました。 (2017/7/22)
解説
  • 様々なサービスでパスワードが利用されていますが、安全に運用するのはとても大変です。
    • サービスごとに異なるパスワードを利用することが推奨されていますし、 頻繁にパスワード変更を求めるシステムもありますが、 長い強力なパスワードを頻繁に考えて覚えておくことは不可能です。
    • パスワードを書き留めておくのは安全ではありませんし、書いた場所を忘れてしまうこともあります。
    • 複数のパスワードを管理するシステムが沢山提案されていますが、 管理のための強力な「マスターパスワード」を記憶する必要があり、 マスターパスワードを忘れたり流出させたりすると悲惨なことになります。
  • EpisoPassは以下の方法でこのような問題を完全に解決します。
    • 個人的なエピソード記憶に関連したなぞなぞ問題集を使う
    • 答の選択にもとづき、ユーザが与えた文字列(シード文字列)を変換して文字列を生成する
    • 正しい答を選んだときに生成される文字列をパスワードとして利用する
  • EpisoPassを使うと、パスワードを全く覚えることなく、パスワードに満ちた世界を楽に生きることが可能になります。
使い方
  • https://EpisoPass.com/exampleのようなページでなぞなぞ問題を編集します。
    • 「example」のかわりに好きな名前を利用できます。
  • 問題や答をクリックすると編集できます。
  • +/- ボタンを押すと問題や答の数を増減できます。
  • 適当なシード文字列(パスワード生成の「種」になる文字列)をSeed:欄に入力し、正しい答を選択したとき生成されるパスワードを利用します。
  • https://EpisoPass.com/example/seedstringのようにシード文字列をURLで指定することもできます。
  • 既存のパスワードを利用したい場合は、Password:欄に現在のパスワードを入力するとシードが逆生成されます。
  • 「サーバにセーブ」ボタンを押すと 問題 / 選択枝 / シード がサーバにセーブされます。パスワードはセーブされません。
  • 「ファイルにセーブ」ボタンを押すと 問題 / 選択枝 / シード が qa.json というJSONファイルにセーブされます。
  • 「Androidアプリ」ボタンを押すと Androidアプリがビルドされてダウンロードできます。
  • JSONファイルを問題画面にDrag&Dropするとデータをアップロードできます。
特徴
  • シード文字列となぞなぞ問題集があればいつでもパスワードを生成できるので パスワード文字列を覚える必要がありません。
  • 強力なパスワードを作るために頭を使う必用がありません。 長いシード文字列を使えば強力なパスワード文字列を生成できます。
  • サービスごとに異なるシード文字列を使うことにより、 様々なパスワードを自由に生成できます。
  • サービスごとに回答を変えることによっても様々なパスワードを生成できます。
  • HTMLをローカルマシンにセーブしておけばネット接続しなくてもパスワードを生成できます。
  • Androidアプリを利用することもできます。
  • FAQも参照ください。

その他
  • なぞなぞ問題は沢山作っておくと安全です。
  • なぞなぞの答を他人に言ってはいけません。 他人に話す可能性が低い問題を作ればいいでしょう。
  • 正答がわからなければパスワードを生成できませんから、 なぞなぞ問題や選択枝の数が充分多ければシード文字列やなぞなぞ問題集を公開しても大丈夫でしょう。
