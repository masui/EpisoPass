EpisoPass らくらく問題作成

人名や地名のリストから簡単にEpisoPass問題を作ることができます。

1. 名前リストを作成

  • 思い出がある人名や地名を30行ぐらい並べて下さい。
  • 適当な名前を混ぜてもかまいません。

2. 質問リストを作成

  • 上のリストのうち10個程度に対し、それを特定できる質問を並べて下さい。
  • 他人が知らない秘密の事実で、将来忘れないようなエピソードを書いて下さい。
  • 質問の例

3. EpisoPass登録

  • ID と シード文字列 を指定して 登録 ボタンを押すと EpisoPass編集ページ ができます。
    • ID :   シード文字列 :   登録
  • IDはEpisoPass問題を識別するための名前です
  • 「Masui_Twitter」というIDの場合https://EpisoPass.com/Masui_TwitterというURLになります。
  • シード文字列はパスワード文字列を計算する「種」となるもので、生成されるパスワードにはシード文字列の文字種と長さが反映されます。

4. DASパタン登録

  • EpisoPassのDAS(Draw-a-Secret)機能を使うと解答選択が簡単になります。
  • EpisoPass編集ページで正答を選択してから 画面右上の「DAS」ボタンを押すとDASパタンを登録できます。
  • DASパタンを登録すると https://EpisoPass.com/Masui_Twitter.html のような EpisoPass利用ページ でDASを利用できるようになります。
